# Git-story-# 🚀 CommitVerse

Turn messy Git commits into meaningful project stories.

## ✨ Features
- AI-powered commit storytelling
- Planet-based visualization
- Timeline view of commits
- Feature / Fix / Milestone classification

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- GitHub API (planned)
- AI integration (planned)

## 🚀 How to Run
Open `index.html` in browser

## 💡 Idea
We transform raw Git history into human-readable narratives for better understanding and collaboration.
